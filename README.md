# 🔐 VALCRO Password Generator — Beast Mode v2.0

VALCRO is an advanced terminal-based password and key generation toolkit built in pure Python.

Designed for security enthusiasts, developers, and power users who want **maximum entropy**, **full control**, and **zero external dependencies**.

---

## ⚡ Features

- Multiple password generation modes
- Passphrase generator (human-memorable + high entropy)
- Pronounceable password engine
- Pattern-based password creation
- PIN / OTP generator
- Cryptographic key generator
- Password strength analyzer
- Entropy & crack-time estimation
- Export to TXT / CSV / JSON
- Session history tracking
- ANSI styled interactive UI
- Works on Linux, Windows, Termux & Pydroid
- No external libraries required

---

## 🧠 Generation Modes

- Standard secure passwords
- Passphrases
- PIN / OTP
- Pattern generator
- Pronounceable passwords
- Bulk mixed generation
- Password analyzer
- Crypto key generator

---

## 🔒 Security

- Uses Python `secrets` module (cryptographically secure RNG)
- Entropy-based strength scoring
- Crack-time estimation
- SHA256 fingerprint preview
- No telemetry
- Offline & privacy friendly
